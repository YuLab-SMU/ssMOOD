# ssMOOD


## 前端配置
Vue 3.5 Beta版本
#### 安装项目依赖
```
npm install
```

#### 运行开发服务器
```
npm run serve
```

#### 构建
```
npm run build
```


## 后端配置
LAMP环境：
```
ubuntu 20
apache2
mysql 5.7 或 MariaDB 10.3（不支持mysql 8）
php 7.x
```

#### PHP配置
需要将php的内存限制提高到512MB或者在脚本中设置，当前采用的是通过脚本设置。在php脚本第二行添加：
```
ini_set('memory_limit', '512M');
```
